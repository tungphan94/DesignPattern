# デザインパターン
デザインパターンは、 ソフトウェアの設計でよく起きる問題に対する典型的な解決方法です。
パターンの一つ一つは、 自分のコードの設計の一つ一つは、 自分のコードの設計上の問題に合わせて調整可能な設計図のようなものです。
